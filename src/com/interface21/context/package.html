This package builds on the beans package to add support for message sources and for the Observer design pattern, and the ability for application objects to obtain resources using a consistent API.

There is no necessity for Spring applications to depend on ApplicationContext or even BeanFactory functionality explicitly. One of the strengths of the Spring architecture is that application objects can often be configured without any dependency on Spring-specific APIs. 这个包构建在beans包之上，以增加对它的支持 消息源和Observer设计模式，以及 应用程序对象使用a获取资源的能力 一致的API。

Spring应用程序无需依赖 在ApplicationContext甚至BeanFactory功能上 明确。 Spring架构的优势之一 是应用程序对象通常可以配置没有 对Spring特定API的任何依赖。
